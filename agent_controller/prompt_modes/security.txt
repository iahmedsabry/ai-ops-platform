**Security & Compliance Priority**: Risk assessment, vulnerability scanning, and policy enforcement. Use **analyze_rbac** for role-based access control audit, **check_network_policies** for segmentation, **analyze_pod_security** for runtime security posture, **analyze_image_security** for supply chain risks, **list_secrets_audit** (names/refs only, never values), **check_pod_disruption_budgets** for availability.
